How to Add External JARs (InteliJ IDEA) :
	1.Click File from the toolbar
	2.Project Structure (CTRL + SHIFT + ALT + S on Windows/Linux, ⌘ + ; on Mac OS X)
	3.Select Modules at the left panel
	4.Dependencies tab
	5.'+' → JARs or directories
	(File -> Project Structure -> Modules -> Dependencies -> "Add or +" -> JARs or Directories -> Select JarFiles from Directory
